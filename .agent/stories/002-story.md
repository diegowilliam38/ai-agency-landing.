# Story 002: Upgrade Visual para WebGL 3D (Mira) e Seções de AI Engineering

## Metadata
- **ID:** 002
- **Title:** Migração para WebGL (Mira 3D) e Adição de Seções de Arquitetura
- **Status:** Done
- **Owner:** @sm

## Context
A landing page original foi criada focando em animações 2D. Para atingir um público corporativo de alto nível (CTOs, Diretores de Dados) e se posicionar como uma boutique de AI Engineering, decidimos aplicar a engine "Mira 3D" (baseada em WebGL e React Three Fiber). Além do visual, identificamos a necessidade de provar autoridade técnica adicionando seções de arquitetura de software, stack de dados e governança.

## User Story
**Como** visitante de perfil executivo/técnico (CTO, Engenheiro de Dados),
**Eu quero** ver topologias de dados em formato 3D e entender claramente qual a stack tecnológica da agência,
**Para que** eu sinta confiança de que a agência não faz apenas "chatbots", mas domina governança, observabilidade e Data Engineering em nível enterprise.

## Acceptance Criteria
- [x] O visual 2D (D3.js) deve ser totalmente substituído por WebGL (Three.js).
- [x] O slide "Jurídico" deve ter uma topologia de cofre/escudo (Zero Trust).
- [x] O slide "B2B" deve ter uma topologia de funil de alta velocidade.
- [x] O slide "EdTech" deve ter uma topologia de rede neural.
- [x] Deve existir uma seção documentando a Stack (LangChain, Pinecone, Next.js).
- [x] Deve existir uma seção detalhando Governança (Zero Trust, LGPD, LangSmith).
